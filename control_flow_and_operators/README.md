# Operadores e controle de fluxo

Este tópico cobre operadores aritméticos, lógicos e bitwise, além de `if`, `switch`, `for`, `break` e `continue`.

## Executar

```bash
go run .
```

## Testar

```bash
go test ./...
```

## Pontos-chave

- `switch` sem expressão funciona como cadeia de condições.
- `for` é o único laço da linguagem.
- Bitwise é útil para flags e operações de baixo nível (`1 << 3`).
- Labels devem ser usadas com critério para manter legibilidade.
