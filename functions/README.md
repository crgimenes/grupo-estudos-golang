# Funções

Este tópico cobre parâmetros, múltiplos retornos, variádicas, closures, função como valor e `init`.

## Executar

```bash
go run .
```

## Testar

```bash
go test ./...
```

## Pontos-chave

- Retorno `(valor, erro)` é padrão em Go.
- Funções variádicas aceitam zero ou mais argumentos.
- Closures capturam variáveis do escopo externo.
- `init` roda antes de `main`.
