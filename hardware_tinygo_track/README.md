# Trilha de hardware e embarcados

Este tópico classifica o conteúdo legado de hardware para TinyGo/automação física.

## Reintegrado para trilha dedicada

- `nao_tratados/serial`
- `nao_tratados/pwm`
- `nao_tratados/botao_e_LED`
- `nao_tratados/ras`

## Decisão

- manter em trilha separada por depender de hardware real e toolchain específica;
- preservar exemplos como referência didática;
- priorizar execução em ambiente controlado com documentação de pinos e sistema operacional.
