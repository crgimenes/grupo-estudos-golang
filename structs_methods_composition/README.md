# Structs, métodos e composição

Este tópico cobre structs, métodos com receiver por valor e por ponteiro, e composição com embedding.

## Executar

```bash
go run .
```

## Testar

```bash
go test ./...
```

## Pontos-chave

- Receiver por valor não altera o original.
- Receiver por ponteiro permite mutação.
- Embedding compõe comportamento sem herança clássica.
